## 使用方法

global

```sh
npm install --global eslint prettier @xwtec/eslint-config eslint-config-prettier eslint-plugin-prettier
npx node-wget https://github.com/xwtec/dotfiles/raw/master/prettier/prettier.config.js
npx node-wget https://github.com/xwtec/dotfiles/raw/master/eslint/.eslintrc.js
eslint **/*.js
```

project

```sh
yarn add --dev eslint prettier @xwtec/eslint-config
npx node-wget https://github.com/xwtec/dotfiles/raw/master/prettier/prettier.config.js
npx node-wget https://github.com/xwtec/dotfiles/raw/master/eslint/.eslintrc.js
npx eslint **/*.js
```
